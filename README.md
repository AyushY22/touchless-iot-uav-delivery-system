# touchless-iot-uav-delivery-system
Touchless IoT-based UAV system for secure small parcel delivery with mobile app control and real-time analytics

This repository presents the complete implementation of the research project **“Touchless IoT-Based UAV System for Small Parcel Delivery via Mobile Application.”** The proposed system is designed to modernize last-mile delivery by combining **Unmanned Aerial Vehicle (UAV) technology, Internet of Things (IoT), and mobile computing** to achieve secure, smart, and contactless parcel transportation.

**Live Web Application Demonstration**
Web App Link: https://uavcontrollerapp.web.app/

**Core Objectives of the Research**
- To develop a smart and touchless parcel delivery model using UAV technology  
- To implement **gesture-based UAV control** for intuitive drone navigation  
- To integrate **voice-based UAV control** for hands-free operation  
- To connect the system with IoT for real-time communication between app and drone  
- To reduce human involvement in parcel handover operations  
- To improve delivery efficiency, speed, and safety in urban areas  
- To provide secure parcel access through OTP-based authentication  
- To create a practical mobile-based control system for users

**System Modules**
- **Mobile Application:** User interface for tracking, gesture and voice control, and verification  
- **Cloud Database:** Stores commands, delivery data, and live updates  
- **UAV Controller:** Executes flight, navigation, and delivery operations  
- **Gesture Recognition Module:** Detects user gestures and converts them into UAV commands  
- **Voice Recognition Module:** Processes spoken instructions for UAV movement  
- **Authentication Layer:** Ensures secure parcel handover through OTP verification  
- **Analytics Dashboard:** Tracks system performance and delivery records

 **Key Features Implemented**
- Real-Time UAV Tracking  
- Gesture-Controlled UAV Navigation  
- Voice-Controlled UAV Commands  
- IoT-Based Live Communication  
- Touchless Parcel Handover  
- OTP Verification System  
- Fast and User-Friendly Interface  
- Scalable Smart Delivery Architecture

**Research Significance -**
This work demonstrates how emerging technologies can be integrated to solve real-world logistics challenges. By combining UAV automation, IoT connectivity, gesture interaction, and voice intelligence, the proposed system provides a next-generation delivery framework suitable for smart cities, healthcare logistics, emergency supply transport, and e-commerce applications.
